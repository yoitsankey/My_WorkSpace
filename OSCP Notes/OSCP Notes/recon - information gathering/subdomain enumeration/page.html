- subfinder - subfinder -d target.com -o output_filename.txt -v - creating automated list by ourselves using kali tools
  • cat output_filename.txt | wc -l = gives total number of items in a text list.
  • sort output_filename.txt | uniq -d = gives a list of duplicates
  • sort output_filename.txt | uniq > duplicates_removed_newfile.txt = Removed duplicates, anew tool - cat list1.txt list2.txt | anew > combinedlist.txt - can also be used to remove duplicates and combine two text lists.
