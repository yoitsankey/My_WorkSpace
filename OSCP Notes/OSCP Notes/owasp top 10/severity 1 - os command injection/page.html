• Occurs when server-side code (like PHP) in a web application makes a system call to a function that interacts with the server's console directly.
  • An injection web vulnerability allows an attacker to take advantage of that call to execute operating system commands arbitrarily on the server.
  • Active command injection will return the response to the user. It can be made visible through several HTML elements.
  • Blind command injection does not return the response to the user.
